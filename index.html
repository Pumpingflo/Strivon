Garmin Sync
Einmal einrichten
Strivon sendet deinen Trainingsplan an den Strivon-Sync-Server (z. B. Render). Dort wird mit deinen Garmin-Daten angemeldet und die Einheiten in Garmin Connect angelegt. Du musst in Strivon angemeldet sein — der Server prüft dein Login über Supabase.
⚠ E-Mail, Passwort und Token werden nur in deinem Browser (localStorage) gespeichert — nicht auf öffentlichen Rechnern. Server neu deployen hilft oft, wenn Garmin den Login geändert hat (garminconnect>=0.3.2). Ohne Token: 2FA am Garmin-Konto deaktivieren oder separates Konto ohne 2FA.
